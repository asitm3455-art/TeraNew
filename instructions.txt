Create a fully working Python Telegram bot using Pyrogram with the folder structure below.  
This bot must download Terabox links using this API:
https://teraapi.boogafantastic.workers.dev/play?url=YOUR_LINK
 
### BOT FEATURES ###
 
1. LOG_CHANNEL → save new user join logs  
2. ERRORS_CHANNEL → forward failed download link + user info  
3. STORAGE_CHANNEL → send all downloaded files + user info  
4. Auto-Rename Feature →  
   Users can set default filename prefix (saved in mongodb)  
5. Anti-Spam + Rate Limit  
6. FloodWait handling  
7. Progress bar during uploading  
8. Direct link extractor + file name fetcher  
9. Downloading via direct API URL  
10. Custom thumbnail support for uploaded files  
11. Start, Help commands  
12. Broadcast message system for owner  
13. Permanent user database (MongoDB)
